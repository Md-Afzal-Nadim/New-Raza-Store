# 🛒 New Raza Store – Grocery Ordering Website

A fully responsive online grocery store website built using **HTML, CSS, and JavaScript**.  
This project is designed for local customers to easily browse products, add items to cart, and place orders directly through **WhatsApp automation**.

🚀 Live website is deployed on **Vercel** and indexed with **Google Search Console**, so users can find it by searching **New Raza Store** on Google.

## ⚙️ How It Works

1. User visits website  
2. Selects grocery category  
3. Adds products to cart  
4. Clicks Order Button  
5. WhatsApp opens automatically  
6. Product names + order details sent directly to store owner
